Overview

This documentation provides a step-by-step guide for installing, configuring, and upgrading the software. It also includes server requirements, database setup, file upload instructions, login credentials, and support information.

Server Requirement

Laravel requires certain minimum PHP versions and extensions. Make sure your web server supports the following:

    Minimum PHP Version:

  • PHP >= 8.1

    • Required PHP Extensions:

  • cURL PHP
  • DOM PHP
  • Fileinfo PHP
  • Filter PHP
  • Hash PHP
  • Mbstring PHP
  • OpenSSL PHP
  • PCRE PHP
  • PDO PHP
  • Session PHP
  • Tokenizer PHP
  • XML PHP

How to Create Database

Step 1: Access the Database Creation Tool

First, navigate to the Databases section of your hosting panel. Click on MySQL® Databases to begin the process. This tool allows you to create and manage your databases and database users.

Step 2: Create a New Database

On the next screen, you will be able to create a new database. You will see a field to enter the name of your new database. The hosting provider automatically adds a prefix to the database name for security. After entering the name, click Create Database.

Step 3: Create a New User

After creating the database, you'll need to create a user for it. Scroll down on the same page to the Add New User section. Enter a Username and a strong Password in the provided fields. Use the Password Generator tool to create a secure password. Click Create User to save the new user.

Step 4: Add the User to the Database

Once the user is created, you must associate them with the database. Scroll down to the Add User To Database section. From the dropdown menus, select the User you just created and the Database you want to link them to. Click Add to proceed.

Step 5: Assign Privileges

The final step is to grant the new user permissions to manage the database. On the Manage User Privileges page, check the box for ALL PRIVILEGES. This gives the user full control over the database, allowing them to create, modify, and delete tables and data. Click Make Changes to finalize the process. Your database and user are now set up and ready to use.

How to Upload Files

Step 1: Access the File Manager

Log in to your cPanel account. In the main menu, look for the Files section and click on File Manager. This will open the file manager interface in a new tab.

Step 2: Navigate to the Correct Folder

In the File Manager, navigate to the folder where you want to upload your files. For a primary domain, this is typically the public_html folder. This folder is the "web root" and contains all the files that are accessible to the public on your website.

Step 3: Select and Upload Files

Once you are in the correct directory, click the Upload button in the toolbar at the top of the page. This will open a new window where you can drag and drop your files or click Select File to browse for files on your computer.

Step 4: Verify the Upload

After the upload is complete, you should see a 100% progress message. The uploaded file should now appear in the folder you selected in the File Manager.

How to Install

Step 1: Start the Installation

Open a web browser and navigate to your website's installation URL, which is typically yourdomain.com/install. This action will initiate the installation process and lead you to the App Installation Process page.

Step 2: Check Server Requirements

The next step is to ensure your server meets all the necessary requirements for the application. Click the "Click to Check Your Server Requirements" button to proceed.

Step 3: Verify App Permissions

After confirming the server extensions, the installer will check the necessary directory permissions. Ensure all permissions are correct (often shown with a green checkmark next to them) and then click "Click to Configure Your App Environment".

Step 4: Configure App Environment

You will be given options to configure the app's environment. For a straightforward installation, select "Using Form Wizard Setup".

Step 5: Enter App and Database Details

This is a critical step where you link the application to your database. Fill in the required fields with the information you gathered when setting up your database. This includes the Database Host, Database Name, Database User Name, and Database Password. You will also need to enter an Application Name and Application Url. Once all the details are entered, click "Click to Install".

Step 6: Finalize Installation

The installer will now run the installation process, which includes preparing the database and running migrations. Once you see the "Application has been successfully installed" message, you have successfully completed the process. Click "Click here to exit" to finish.

Software Upgrade Process

Before proceeding with the update, ensure that you have completed a full backup of your software.
Important Notes:
1. Language Files Modifications:
  • If you have modified any language files, your changes will be overwritten. The new version will come with updated language files, and your previous customizations in resources/lang will be lost.
  • It's highly recommended that you back up your edited language files for future reference.
2. File Modifications:
  • Any changes made to the software's core files will be lost after the update. Ensure that you have backed up any customizations or modifications.
3. Disclaimer:
  • We are not responsible for any data loss that occurs during a manual update. Please ensure that you follow the backup instructions carefully.
Backup Instructions:
1. Files to Back Up:
  • .env file
  • .htaccess file
  • storage directory
2. Update Process:
  • Upload and replace all the files & folders on your server with the contents of the main_file folder from the new version.
  • Restore your .env, .htaccess, storage files and folder back onto your server.
  • Log in to the software and follow the steps in the update wizard to complete the process.
